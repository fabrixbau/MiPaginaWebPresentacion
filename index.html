Experiencia laboral

Durante más de un año, he dedicado mi tiempo al estudio y desarrollo de proyectos personales y profesionales, destacándome en la creación de páginas web y herramientas de automatización. Mis proyectos incluyen el desarrollo de sitios web para una cafetería y un restaurante, así como la creación de mi propio sitio web como plataforma de presentación profesional. He realizado dos proyectos de scrapping significativos: uno para extraer y gestionar datos de inmuebles en Mercado Libre y otro para documentar el progreso de un proyecto en desarrollo, integrando funcionalidades con la API de Google Sheets. Además, he diseñado tres APIs RESTful para diferentes aplicaciones, incluyendo la gestión de citas médicas, el manejo de usuarios y un blog interactivo. Mi experiencia se extiende al uso de Websockets para simular transacciones comerciales en tiempo real y para desarrollar un chat en línea, demostrando mi habilidad para trabajar con tecnologías de comunicación avanzadas.
